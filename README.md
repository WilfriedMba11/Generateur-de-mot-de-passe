# Générateur et Vérificateur de Mot de Passe 

Un petit programme en Python permettant :
- de **générer des mots de passe sécurisés**,
- de **tester la robustesse** d’un mot de passe existant,
- et de donner des **conseils pour l’améliorer**.

---

##  Fonctionnalités
- Génération de mots de passe aléatoires avec lettres, chiffres et caractères spéciaux.
- Vérification de la force d’un mot de passe selon :
  - Longueur
  - Majuscules
  - Minuscules
  - Chiffres
  - Caractères spéciaux
- Évaluation du niveau de sécurité : **Faible**, **Moyenne**, ou **Forte**.
- Interface en ligne de commande avec menu interactif.
